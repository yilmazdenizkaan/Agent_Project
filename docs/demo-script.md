# Demo Script

1. Open BugCourt AI and paste a repository URL into the input field.
2. Click **Analyze Repository** to simulate a code review.
3. The dashboard displays generated court cases, starting with “Possible Authentication Bypass.”
4. Click a case to inspect evidence, prosecutor and defense arguments, expert witness notes, judge verdict, confidence score, and recommended fix.
5. Explain how BugCourt AI turns static findings into courtroom-style debate and judge decisions.
6. Conclude by noting the safe MVP design: no repository execution, only static analysis and grounded evidence.
