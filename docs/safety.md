# Safety

BugCourt AI is built as a safe static analysis MVP with strict boundaries.

- No repository code is executed or imported at runtime.
- The system uses static analysis placeholders only.
- Sensitive values and secrets are not processed or exposed.
- Every court case is grounded in visible code evidence or a static finding.
- Agents must not invent vulnerabilities without evidence.
- The MVP is intentionally limited to mock data and API routes for safe demo use.
